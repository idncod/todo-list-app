Prerequisites
Node.js >= 18

pnpm / npm / yarn

Installation
```bash
git clone https://github.com/idncod/todo-list-app.git
cd todo-list-app
npm install
npm run dev
```

Testing
```bash
npm run test
```
or in live watch mode - re-runs on file changes
```bash
npm run test:watch
```

or run Vitest manually
```bash
npx vitest
```
